# SelenaArtBot 🌙

Telegram бот для генерации и редактирования изображений через Polza.ai API

## Команды
- /start - начать работу
- /status - статистика
- /buy - купить премиум
- /help - помощь

## Установка
1. `pip install -r requirements.txt`
2. Создать `.env` с токеном
3. `python bot.py`
